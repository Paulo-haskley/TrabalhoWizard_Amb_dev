# TrabalhoWizard_Amb_dev

Problemas:
Muito uso de variaveis incorretas e dificifeis de se decifrar, juntamente com uma grande repetição de numeros magicos repetidos durante o codigo.



Decisões:
Uso de constantes para quebrar esse grande numero de numeros magicos e a declaração correta dessas variaveis para uma melhor compreensao do codigo
